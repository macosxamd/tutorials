# ü™ü Windows XP / 2003 License Management Guide

### (Remove, Restore, and Change Product Key)

This guide explains how to properly **remove**, **restore**, and **change** a Windows XP or Windows Server 2003 license.
These steps are intended for legitimate system maintenance, activation repair, or hardware migration.

---

## üß© 1. Removing the License (Deactivation)

When activation is corrupted or a new key must be used, you can safely remove the license data.

### **Step-by-step:**

1. **Open Command Prompt**

   * Click **Start ‚Üí Run**
   * Type:

     ```
     cmd
     ```

     Press **Enter**

2. **Reset activation grace period**

   * In Command Prompt, run:

     ```
     rundll32.exe syssetup,SetupOobeBnk
     ```
   * This resets Windows‚Äô activation timer and clears the activation state (but keeps your current installation intact).

3. **Optional: Delete activation files**

   * To completely remove activation data:

     ```
     ren %systemroot%\system32\wpa.dbl wpa.old
     ren %systemroot%\system32\wpa.bak wpa.old
     ```
   * This forces Windows to require activation again on next boot.

---

## üß© 2. Restoring the License (Reactivate from Backup)

If you backed up activation files from a previously activated installation, you can restore them.

### **Before reinstalling or cloning:**

1. Go to:

   ```
   %systemroot%\system32
   ```
2. Copy these files to a safe location (e.g., USB drive):

   ```
   wpa.dbl
   wpa.bak
   ```

### **After reinstalling Windows:**

1. Boot into **Safe Mode** (press **F8** during startup).
2. Copy your backed‚Äëup files back to:

   ```
   %systemroot%\system32
   ```

   and overwrite existing ones.
3. Reboot normally.

   * If the hardware is the same, Windows will appear as already activated.
   * If major hardware has changed, reactivation (online or phone) may still be required.

---

## üß© 3. Changing the Product Key (Without Reinstalling)

You can change the product key directly in Windows without reinstalling.

### **Method 1 ‚Äî Using the Activation Wizard**

1. Open **Run** dialog and enter:

   ```
   oobe/msoobe /a
   ```
2. The **Activate Windows** wizard will open.
3. Choose **Yes, I want to telephone a customer service representative to activate Windows**, then click **Next**.
4. Click **Change Product Key**.
5. Enter your new valid key and press **Update**.
6. Once the key is accepted, click **Remind me later** and reboot.
7. After reboot, run:

   ```
   oobe/msoobe /a
   ```

   * If the message says *‚ÄúWindows is already activated‚Äù*, the key change was successful.

## ‚ö†Ô∏è Notes & Best Practices

* Always back up `wpa.dbl` and `wpa.bak` before making changes.
* The activation data is **hardware-specific** ‚Äî if you change your motherboard, CPU, or hard drive, reactivation will usually be required.
* These methods are for **legitimate maintenance**, **migration**, or **activation repair** ‚Äî not to circumvent licensing.
